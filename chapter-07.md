# 第7章：读取文档

## 核心主题

从 PDF、Office 等二进制文档中提取文本，与网页抓取互补。

---

## 7.1 PDF 文本提取

### 核心概念

- **版式 PDF**：文字以绘制指令存在，可用库抽取  
- **扫描版 PDF**：本质是图片，需 OCR（见第 13 章）  

### 依赖（示例）

```bash
pip install pypdf
```

### 可运行代码示例

`code/chapter07/read_pdf.py`（若未安装 `pypdf`，脚本内会提示）  

### 新手易错点

- PDF 无「段落」概念，抽取结果常缺换行，需后处理（第 8 章）  
- 加密 PDF 需密码参数  

---

## 7.2 Word 等格式

### 核心概念

- **`python-docx`**：读取 `.docx` 段落与表格（与旧版 `.doc` 不同）  

```bash
pip install python-docx
```

## 本章小结

- 文档抓取与 HTML 抓取共享同一后端：**解析 → 清洗 → 结构化**  
- 先判断文档是**文本型还是扫描型**，再选工具链
