## 一、爬虫核心原理（极简版）

**爬虫** = 模拟浏览器访问网页 → 拿到网页源代码 → 提取需要的信息  

**流程：**

1. 发送请求（Python 程序 → 服务器）  
2. 服务器返回 HTML 源码  
3. 解析源码，取出标题、文本、链接等内容  
4. **不会自动加载图片、CSS、JS**（只拿到本次请求的 HTML 文本；除非你再发请求去拉资源）  

---

## 二、必须掌握的两个库

### 1. urllib

- Python **自带标准库**，无需安装  
- 作用：打开网页、获取 HTML 内容（常用 `urllib.request.urlopen`）  

### 2. BeautifulSoup4

- 第三方库，需要安装  
- 作用：把杂乱的 HTML 变成结构化对象，方便提取数据  

安装命令：

```bash
pip install beautifulsoup4
```

---

## 三、HTML 解析器（3 种任选）

创建 `BeautifulSoup` 时必须指定解析器：

1. **`html.parser`（推荐入门）**  
   - Python 自带，无需额外安装  
   - 稳定、够用  
2. **`lxml`**  
   - 速度快，需安装：`pip install lxml`  
3. **`html5lib`**  
   - 容错强，速度较慢：`pip install html5lib`  

---

## 四、最容易出问题的 3 类情况（建议处理）

1. **`HTTPError`**：页面不存在（404）、服务器报错（500）等  
2. **`URLError`**：连不上服务器、断网、DNS/域名错误等  
3. **标签缺失**：`find` 找不到时返回 `None`，若再链式 `.children` 等可能 **`AttributeError`** —— 应先判断再取值  

---

## 五、可直接运行的完整代码（推荐版本）

**文件路径（本仓库）**：`chapter-01/get_title.py`  

实现要点：网络异常 + 解析后安全取 `h1`；请求带 **User-Agent** 与 **timeout**，减少被站点直接拒绝或长时间卡住。

```python
from urllib.error import HTTPError, URLError
from urllib.request import Request, urlopen

from bs4 import BeautifulSoup


def get_h1_text(url: str):
    try:
        req = Request(url, headers={"User-Agent": "Chapter01GetTitle/1.0 (educational)"})
        html = urlopen(req, timeout=15)
    except (HTTPError, URLError, OSError) as e:
        print(f"访问失败：{e}")
        return None

    bs = BeautifulSoup(html.read(), "html.parser")
    h1_tag = bs.find("h1")
    if h1_tag:
        return h1_tag.get_text(strip=True)
    return "页面没有 h1 标签"


if __name__ == "__main__":
    for demo_url in (
        "https://www.pythonscraping.com/pages/page1.html",
        "http://www.pythonscraping.com/pages/page1.html",
    ):
        result = get_h1_text(demo_url)
        if result is not None:
            print("网页 H1 标题：", result)
            break
```

（若你本地 `https` 握手失败，脚本会**自动回退**到 `http`；生产环境优先修复 TLS/代理。）

## 六、本章重点总结

1. 爬虫 = 请求网页 + 解析 HTML + 提取信息  
2. 访问网页建议加**超时**与**异常处理**，脚本更稳  
3. BeautifulSoup 是解析 HTML 的常用工具  
4. 标签可能不存在，**先判断再取值**
