# 第9章：自然语言处理

## 核心主题

对抓取的文本做分词、停用词过滤、词干提取等，为分类或聚类做准备。

---

## 9.1 NLTK 基础流程

### 核心概念

- **分词**：英文 `word_tokenize`；中文常需 `jieba` 等（本书以英文例为主）  
- **停用词**：高频无信息词表过滤  
- **词干 / 词形**：`PorterStemmer` 等减少词形变化带来的稀疏性  

### 依赖

```bash
pip install nltk
```

首次使用需下载数据包（在 Python 中 `nltk.download("punkt")` 等）。  

### 可运行代码示例

`code/chapter09/nltk_basics.py`

### 新手易错点

- NLTK 数据包体积与路径问题在 CI 中需缓存或 mock  
- 中文场景下 NLTK 能力有限，应换合适分词器  

---

## 本章小结

- NLP 是**特征工程**：输出常为词袋、n-gram 或向量，供机器学习使用  
- 抓取文本噪声大时，先清洗（第 8 章）再分词效果更好  

## 本章练习题

1. 对一段英文做分词 + 小写 + 停用词过滤 + 词干化，打印词表  
2. 统计词频前 10，思考哪些词应加入自定义停用表  
3. 阅读 NLTK 文档，试 `pos_tag` 对一句话打词性
