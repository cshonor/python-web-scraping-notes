# 第13章：图像识别与文字处理

## 核心主题

用 Pillow 处理图像，用 Tesseract 等 OCR 从图片中抽取文字（常用于验证码场景外的文档数字化）。

---

## 13.1 Pillow 基础

### 核心概念

- **打开、裁剪、缩放、灰度化**：为 OCR 或特征提取预处理  

```bash
pip install Pillow
```

### 可运行代码示例

`code/chapter13/pillow_resize.py`（若无样例图则生成纯色图演示）  

---

## 13.2 OCR 与伦理边界

### 核心概念

- **`pytesseract`**：Tesseract 的 Python 绑定；需单独安装 Tesseract 可执行文件  
- **验证码**：自动识别验证码常涉及未授权访问，**学习请用自建或授权测试环境**  

### 新手易错点

- OCR 对分辨率、对比度敏感，预处理往往能大幅提升准确率  
- 将 OCR 用于绕过人机验证可能违法或违约  

---

## 本章小结

- 图像流水线：**解码 → 预处理 → OCR → 后处理**  
- 技术能力与**合法使用场景**必须同时考虑
